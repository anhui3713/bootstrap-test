描述

示例

导航条必须使用<nav>标签或role="navigation"的div中, 样式设置为class="navbar navbar-default",
导航条中的内容包含在.navbar中的.container-fluid容器中
将导航条中的logo等内容放置到.navbar-fluid中,用class="navbar-brand"来表示logo.可以替换成自己的图片或文字.
  • 将导航条内放置品牌标志的地方替换为<img>元素即可展示自己的品牌图标。由于 .navbar-brand 已经被设置了内补（padding）和高度（height），你需要根据自己的情况添加一些 CSS 代码从而覆盖默认设置。

  • 在<li>标签上添加class="dropdown"并在其中设置下拉控件来实现下拉菜单,下拉控件的按钮需要设置为class="dropdown-toggle"

非表单操作
  • 一般按钮在按钮上添加navbar-btn类
  • 一般文本在<p>标签上添加class="navbar-text", .navbar-text默认为float: left;,如果要按其他样式显示则需要自定义样式代码来控制.
  • 一般的超链接直接在<a>上添加class="navbar-link"
  • 通过class="navbar-left"和class="navbar-right"来控制导航中的内容对齐

固定在底部的导航: 只需要在<nav>标签上添加.navbar-fixed-bottom样式

  • 因为导航条会浮动到顶部,所以会遮盖顶部部分内容,所以可以通过设置body { padding-top: 70px; } 来将内容往下移动
